# Checklists

> Actionable, lightweight verification steps to run before committing, opening PRs, or releasing software.

## Included Checklists

*   [PRE_COMMIT.md](PRE_COMMIT.md) — Steps to run locally before staging or committing changes.
*   [PRE_PR.md](PRE_PR.md) — Quality gates to review before opening a pull request.
*   [PRE_RELEASE.md](PRE_RELEASE.md) — Pre-deployment verification and environment sanity checks.
*   [PRODUCTION.md](PRODUCTION.md) — Standard production configuration and validation requirements.
*   [SECURITY.md](SECURITY.md) — Hardening, vulnerability checks, and secret scans.
*   [OBSERVABILITY.md](OBSERVABILITY.md) — Logging, telemetry, and metrics verification.
*   [BACKUP.md](BACKUP.md) — Backups and restore capability checks.
