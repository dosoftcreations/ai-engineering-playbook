# Agent Instructions Template

Use this file as a starting point for repository-specific agent guidance.

## Purpose

- Define the expected working style for AI-assisted contributors.
- Keep the scope bounded and production-oriented.
- Emphasize verification and review.

## Suggested rules

- Prefer minimal diffs.
- Do not invent requirements.
- Ask for clarification when the scope is ambiguous.
- Verify changes when evidence is available.
- Call out uncertainty explicitly.
