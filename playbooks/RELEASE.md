# Release Playbook

Use this playbook for planned production releases.

## Release steps

1. Confirm the release scope and success criteria.
2. Complete the pre-release checklist.
3. Verify the critical user flows and security-sensitive changes.
4. Prepare rollback and incident communication plans.
5. Release with monitoring and a clear owner.

## Release exit criteria

A release is ready when the scope is known, the rollback path is clear, and the release can be monitored safely.
