# RFC-0002: Model Routing

## Summary

This proposal recommends assigning models based on task complexity, reliability needs, and cost.

## Motivation

One model size does not fit all engineering tasks.

## Proposal

- use smaller models for targeted edits and repetitive tasks
- use stronger models for reasoning-heavy or risk-sensitive work
- keep human review for production-critical decisions

## Consequences

This approach improves efficiency while preserving quality and safety.
