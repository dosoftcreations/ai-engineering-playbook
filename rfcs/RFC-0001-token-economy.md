# RFC-0001: Token Economy

## Summary

This proposal recommends using token usage as a measurable input to workflow quality rather than as a goal in itself.

## Motivation

Token usage can rise quickly when work lacks clear scope or when the workflow repeatedly revisits the same context.

## Proposal

- keep tasks scoped and bounded
- use the smallest practical model for the task
- preserve human review for high-risk work
- measure token usage alongside outcome quality

## Consequences

This approach encourages more deliberate use of AI while maintaining engineering quality.
