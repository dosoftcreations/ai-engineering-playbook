# RFC-0005: Context Budget

## Summary

This proposal recommends keeping working context bounded to preserve reliability and reduce waste.

## Motivation

Large or poorly scoped context leads to lower-quality reasoning and more rework.

## Proposal

- keep context focused on the current decision or task
- summarize progress instead of re-reading unrelated material
- stop and clarify when scope expands unexpectedly

## Consequences

This improves both quality and efficiency for AI-assisted work.
