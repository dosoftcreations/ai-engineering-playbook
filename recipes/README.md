# Recipes

> Step-by-step exact execution guides for common operational scenarios. 

Unlike **Playbooks** (which explain general principles and workflows), **Recipes** explain the exact commands, scripts, templates, and actions required to execute a specific task in practice. Recipes reference standards and checklists to ensure safety without duplicating their content.

## Included Recipes

*   [CREATE_AGENTS_MD.md](CREATE_AGENTS_MD.md) — Initialize agent rules in a new target project.
*   [FIX_PRODUCTION_BUG.md](FIX_PRODUCTION_BUG.md) — Step-by-step hotfixing of a live production bug.
*   [DATABASE_MIGRATION.md](DATABASE_MIGRATION.md) — Exact steps for executing and testing database migrations.
*   [LAUNCH_SAAS.md](LAUNCH_SAAS.md) — Preparing, staging, and launching a SaaS application.
*   [SHIP_V1.md](SHIP_V1.md) — Releasing and tagging version 1.0.0.
