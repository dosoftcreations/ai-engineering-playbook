# Metrics

> Measurement frameworks designed to assess whether AI-assisted engineering practices are improving outcomes and efficiency.

## Included Metrics

*   [AI_ROI.md](AI_ROI.md) — Calculating engineering output value relative to API token costs.
*   [MANUAL_REWORK.md](MANUAL_REWORK.md) — Tracking bug reopen rates and manual correction requirements.
*   [MODEL_USAGE.md](MODEL_USAGE.md) — Tracking routing cost across model tiers.
*   [REVIEW_QUALITY.md](REVIEW_QUALITY.md) — Assessing PR feedback value and code maintainability.
*   [TOKEN_EFFICIENCY.md](TOKEN_EFFICIENCY.md) — Assessing token savings and budget controls.
*   [AI_TRACKER_TEMPLATE.csv](AI_TRACKER_TEMPLATE.csv) — Reusable spreadsheet template to log tasks, model tiers, durations, token costs, and rework rates.
